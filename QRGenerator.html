📘 เกี่ยวกับ QR Code

QR Code (Quick Response Code) คือรหัสแท่ง 2 มิติที่พัฒนาโดยบริษัท Denso Wave จากญี่ปุ่น เมื่อปี 1994 สามารถเก็บข้อมูลได้หลากหลายรูปแบบ เช่น URL, ข้อความ, ข้อมูลติดต่อ, Wi-Fi และอื่นๆ

  • สแกนง่าย: ใช้กล้องสมาร์ทโฟนหรือแอปสแกน QR อ่านได้ทันที
  • จุข้อมูลได้มาก: เก็บตัวอักษรได้สูงสุด 4,296 ตัว
  • ทนต่อความเสียหาย: ระดับ Error Correction ช่วยให้อ่านได้แม้ส่วนหนึ่งเสียหาย
  • ใช้งานได้หลากหลาย: การตลาด, ชำระเงิน, เช็คอิน, เมนูร้านอาหาร

💡 ทิป: ใช้ Error Correction ระดับ H (30%) หากต้องการใส่โลโก้ทับกลาง QR Code
